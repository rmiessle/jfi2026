# Gettysburg College January Faculty Institute 2026
Workshop for Wrangling Humanities Data with R and AI.

This workshop will cover two parts:

1. Analyzing and visualizing transcribed correspondence from a CSV file
2. Analyzing an OCR-ed corpus from TXT files

- Structured data activity (WWI letters): https://mybinder.org/v2/gh/rmiessle/jfi2026/HEAD?urlpath=/doc/tree/structured-data.ipynb
- Unstructured data activity (Gettysburgian articles): https://mybinder.org/v2/gh/rmiessle/jfi2026/HEAD?urlpath=/doc/tree/unstructured-data.ipynb
